# Atividade: Media Queries com HTML e CSS

## Resumo
Esta atividade tem como foco a construção de uma tela de login responsiva, aplicando conceitos fundamentais de **HTML** e **CSS**, com destaque para o uso de **media queries**.

## Objetivo
Desenvolver uma interface que se adapte a diferentes tamanhos de tela, reorganizando os elementos no mobile para manter legibilidade, proporção visual e boa experiência de uso.

## Conceitos trabalhados
- Estrutura semântica com HTML (`header`, `main`, `section`, `form`)
- Organização de layout com Flexbox
- Estilização de componentes (tipografia, espaçamentos, cores e botões)
- Responsividade com `@media` (quebra de layout para telas menores)
- Ajuste de larguras fixas para dimensões fluidas no mobile

## Resultado esperado
- No desktop: layout em duas áreas principais (apresentação e login)
- No mobile: header escuro no topo e formulário de login abaixo
- Botão e campos ajustados proporcionalmente para telas menores

## Estrutura do projeto
- `src/index.html`: marcação da página
- `src/css/styles.css`: estilos principais e regras responsivas
- `src/css/output.css`: estilos adicionais/gerados

## Como visualizar
1. Abrir o projeto no VS Code.
2. Executar com uma extensão como **Live Server** ou abrir `src/index.html` no navegador.
3. Testar em diferentes larguras de tela usando o modo responsivo do navegador.
